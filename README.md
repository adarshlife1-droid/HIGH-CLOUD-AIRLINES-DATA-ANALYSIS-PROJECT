# HIGH CLOUD-AIRLINES-DATA -ANLYSIS-PROJECT
 The High CLoud Airlines Project is a comprehensive airline performance analysis designed to evaluate operational efficiency, revenue performance, flight trends, and customer travel patterns.
This project analyzes airline data to uncover insights related to flight performance, passenger load factor, route distribution, revenue generation, and delay patterns. The goal is to transform raw aviation data into actionable business intelligence that can support strategic decision-making.
# Project Objectives
Analyze overall airline performance
Track revenue trends over time
Measure Load Factor (Seat Utilization)
Identify high-performing routes
Evaluate delay patterns
Understand seasonal travel demand
Build KPI metrics for airline management
# Dataset description
The dataset includes the following key fields:
Flight Date
Flight Number
Source Airport
Destination Airport
Distance Group
Available Seats
Transported Passengers
Revenue
Load Factor
Aircraft Type
 The data is structured to support multi-level analysis including route-level, monthly, and yearly trends. Key Performance Indicators (KPIs)
The following KPIs were created to measure airline performance:
 Total Flights
 Total Passengers
Total Revenue
 Load Factor (%)
Top Routes by Revenue
Monthly Revenue Trend
Distance Group Analysis
 Top Performing Aircraft
